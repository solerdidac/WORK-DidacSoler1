Diferencias:

Imagenes SVG:

  • Las imagenes SVG se pueden escalar sin la perdida de calidad
  • Van muy bien para representar pequeñas ilustraciones...
  • El archivo tiene poco peso de GB.

Imagenes de Bits:

  • Las imagenes de bits nos proporcionan una resolucion fija
  • Sirven para fotografias e imagenes con detalle
